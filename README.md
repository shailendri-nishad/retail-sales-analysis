# Retail Sales Dashboard Project

## Project Overview

This project analyzes retail sales performance across regions, customer segments, states, and product categories using Python and Power BI. The goal is to transform raw sales data into actionable business insights through data cleaning, analysis, and interactive dashboarding.

## Project Files

- **01_data_cleaning.ipynb** – Data cleaning and preprocessing
- **02_superstore_cleaned.csv** – Cleaned dataset
- **Retail_Sales_Dashboard.pbix** – Interactive Power BI dashboard file
- **Retail_Sales_Dashboard_Screenshot.png** – Dashboard preview
- **INSIGHTS.md** – Key business insights

## Dataset Information

The dataset contains retail sales transactions from **2014–2017**, including:

- Customer details
- Order information
- Product categories and sub-categories
- Sales, profit, and discount data
- Regional and state information

## Data Cleaning Process

- Removed duplicate records
- Converted date columns into proper datetime format
- Standardized column names and categories
- Created derived columns such as Year, Quarter, Month and Days to Ship
- Prepared data for dashboard reporting

## Key Metrics

- **Total Orders:** 5,009
- **Total Sales:** $2.30M
- **Total Profit:** $286K
- **Profit Margin:** 12.5%

## Business Questions

- Which region generates the highest sales?
- Which customer segment contributes the most revenue?
- Which states drive the highest sales?
- Which product sub-categories perform best?
- How has sales performance changed over time?

## Dashboard Features

- KPI Cards
- Regional Sales Analysis
- Customer Segment Analysis
- Top States Analysis
- Sub-Category Performance
- Year-wise Sales Trend
- Interactive Year and Category Filters

## Key Takeaway

The business achieved consistent growth from 2014–2017, with the West region, Consumer segment, and Phone category contributing significantly to overall sales performance.
